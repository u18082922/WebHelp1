4.2.9.2 Add Vehicle Model

Description

This page allows the user to be able to add a new vehicle model. The user enters the model details in the relevant input boxes and the system saves the details in the database.

 

Reference:

Control:

Functionality Description:

1

Vehicle Model Input Box

Click here to enter the vehicle model.

 

2

Add button

This button saves the new added vehicle model details to the system database. The system will redirect to the vehicle model homepage

3

Return to list button

This button cancels the operation and returns the user to the vehicle model home page.

4

Username Link

Click here to sign out of the website

5

Go to Help link

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

•         Possible Error Messages: Vehicle model input box → Vehicle model is required
