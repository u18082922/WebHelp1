4.2.2.4 Generate Policy Report

Description

This page allows the user to generate a quote report whereby the user can view the requested quotes by clients.

Reference:

Control:

Functionality Description:

1

Policy Report option link

Click here to view the Policy Report Page

2

Date Range datetime picker

This allows the user to be able to select the period for which they want the report to be generated for, the start date and end date for the report.

3

Generate Report button

Click here to generate the policy report. This generates the report and displays it on the screen.

4

Download pdf button

 

Click here to download the pdf version of the report. This downloads the report and the report download can be seen at the bottom of the web browser (depending on the browser being used).

 

5

Username Link

Click here to sign out of the website/View profile

6

Go to help

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.
