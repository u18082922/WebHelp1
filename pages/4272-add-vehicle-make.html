4.2.7.2 Add Vehicle Make

Description

This page allows the user to be able to add a new vehicle make. The user enters the make details in the relevant input box and selects the vehicle type which is associated with  the make entered and the system saves the details in the database.

 

Reference:

Control:

Functionality Description:

1

Enter Make name Input Box

Click here to enter the name of the make.

 

2

Vehicle Type dropdown list

This dropdown contains a list of vehicle types. The function allows the user to be able to add the vehicle type associated with the vehicle make.

3

Add button

This button allows the user to add new vehicle make details to the system database. The system will redirect to the vehicle make homepage

4

Return to list button

This button cancels the operation and returns the user to the vehicle make home page.

5

Username link

Click here should the user want to sign out or view user profile.

6

Go to help link

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

•         Possible Error Messages: Enter Make name input box → Name of make is required

      Vehicle Type dropdown list → Vehicle type is required
