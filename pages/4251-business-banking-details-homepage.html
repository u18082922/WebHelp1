4.2.5.1 Business Banking Details Homepage

Description

This page is the home screen for the business banking details, on this page the user can add new business banking details, search, update and delete the business banking details.

 

 

 

 

Reference:

Control:

Functionality Description:

1

Business Banking Details link

Click here to navigate to the business banking details homepage.

2

Add Details Button

Click here add new business banking details. System will redirect to Add Details page.

3

Search bar

This allows for the user to be able to search for the desired bank.

4

Delete button

This button allows the user to delete the specific business banking details next to it.

5

Update button

This button allows for the update of the respective business banking details. System will redirect to update business banking details home page

6

Show number of entries list

This allows for the filtering of only showing a certain number of entries

7

Go to Help link

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

8

Username link

Click here to view profile or to sign out of the system.
