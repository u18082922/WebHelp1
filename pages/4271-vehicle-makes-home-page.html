4.2.7.1 Vehicle Makes Home Page

Description

This page allows the user to be able to view the makes which are saved in the database. The user can then proceed to add more vehicle makes, search, update or delete the makes.

 

 

Reference:

Control:

Functionality Description:

1

Vehicles drop down link *located at the dashboard

This vehicle dropdown list contains the vehicle make, vehicle type, vehicle colour, vehicle model and vehicle series. Click here to view the vehicle information from the dropdown list.

 

2

Vehicle Make link

Click here to view the Vehicle Make homepage.

3

Add Vehicle Make

This button allows for the system to redirect to Add vehicle make page whereby the user can add the new vehicle make details

4

Search Bar

The search bar allows a user to enter the details which the user wants to search for.

5

Show a number of entries

 

 

This allows the user to be able to only view a certain number of entries on the grid view.

6

Update button

This button allows the user to update their vehicle makes details. The system will redirect you to the update vehicle make page

7

Delete button

This button deletes the specified business contact details

8

Username Link

Click here to sign out of the website

9

Go to Help

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.
