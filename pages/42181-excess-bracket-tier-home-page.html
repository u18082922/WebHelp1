4.2.18.1 Excess Bracket Tier Home Page

Description

This page allows the user to be able to view the excess bracket tier which are saved in the database. The user can then proceed to add more excess bracket tiers, search, update or delete the excess bracket tiers.

Reference:

Control:

Functionality Description:

1

Excess Bracket link

This link opens the excess bracket home page

2

Add Excess Bracket

This button allows the client to be able to add an excess bracket. System will redirect to the Add excess bracket page.

3

Show a number of entries

 

This function allows the user to be able to filter and show only a certain number of entries on the grid view.

4

Search Bar

This function allows the user to enter the excess bracket they want to search for

5

Update button

This button allows the user to update the respective bracket details. The system will redirect you to the update excess bracket page

6

Delete Button

This button deletes the excess bracket

7

Go to Help link

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.
