4.2.4.2 Add Bank

 

Description

This page allows the user to be able to add a new bank. The user enters the bank details in the relevant input boxes and the system saves the details in the database.

 

Reference:

Control:

Functionality Description:

1

Name of bank Input Box

Click here to enter the name of the bank.

2

Branch Code Input Box

Click here to enter the bank branch code.

3

Add button

This button saves the new banking details to the system database. These changes can be viewed on the banking details home page.

4

Return to list button

This button redirects the user to the bank homepage

5

Username link

Click here should the user want to sign out or view user profile.

6

Go to help link

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

•         Possible Error Messages: Name of bank input box → Name of bank is required

      Branch code input box → Branch code is required
