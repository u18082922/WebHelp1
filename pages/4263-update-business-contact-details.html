4.2.6.3 Update Business Contact Details

Description

This page allows the user to be able to update the pre-existing business contact details. The user enters the contact details in the relevant input boxes and the system saves the details in the database.

 

Reference:

Control:

Functionality Description:

1

Company name input box

Click here to enter the name of the company in the input box.

2

Company email address input box

Click here enter the company email address in the input box.

3

Company phone number input box

Click here to enter the company phone number in the input box.

4

Company website Input Box

Click here to enter the company website in the input box

5

Update button

This button allows for the system to make the change to the details in the database and this update can be viewed on the business contact details home page

6

Return to list button

Click here to cancel operation and return to business contact details home page.

7

Username link

Click here should the user want to sign out or view user profile.

8

Go to Help link

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

•         Possible Error Messages:

o   Company name input box → Company name is required

o   Company email address input box → Company email address is required

o   Company phone number input box → Company phone number is required

o   Company website input box → Company website is required
