4.2.18.3 Update Excess Bracket

Description

This page allows the user to be able to update the pre-existing excess bracket. The user can enter the excess bracket in the relevant input box and system saves the details in the database.

 

Reference:

Control:

Functionality Description:

1

Bracket Tier Input Box

This input box allows a user to enter the bracket value.

 

2

Bracket Amount

This input box allows a user to enter their bracket amount.

3

Update button

This button saves the bracket value details to the database.

4

Return to list button

This button returns the user to the excess bracket homepage.

5

Go to Help

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

 

•         Possible Error Messages: Bracket value input box → Excess Bracket is required
