4.2.1.1 Login page

 

Description

This page allows the user to login in to the InsureTech website. The user needs to navigate to the input boxes in order to enter their username and their password. The system then verifies the entered details from the data which is contained in the database. The user will then be redirected to the system dashboard screen.

Reference:

Control:

Functionality Description:

1

Enter Username Input box

The username is typed in the input box.

 

Tip*

Username is your email address

2

Enter Password Textbox

Click here to type the password in the textbox. The password needs to be a password which is already created and matched with the username when the client was registering

 

 

Note*

Password is case sensitive

3

Forgot Password link

This function allows the user to be able to create a new password if they forgot their current one.  The user clicks here to redirect to the reset password page.

 

4

Not yet Registered link

This link allows the user to be able to register and gain access to the system.

5

Login

Click here to login to the system. This button allows the user access to the system and logs them in.

6

Go to Help

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

 

•         Possible Error Messages:

o   Incorrect username/password
