4.2.11.3 Update Vehicle series

Description

This page allows the user to be able to update the pre-existing vehicle series. The user can enter the vehicle series name in the relevant input box and select the relevant vehicle series and the system saves the details in the database.

 

Reference:

Control:

Functionality Description:

1

Vehicle series Input Box

Click here to enter the vehicle colour.

 

2

Update button

This button saves the new updated vehicle colour details to the system database. The system will redirect to the vehicle colour homepage

3

Return to list button

This button cancels the operation and returns the user to the vehicle colour home page.

4

Username Link

Click here to sign out of the website/ view the user profile

5

Go to Help link

This link redirects the user to a new tab which opens a document explaining the functions on the page and how they work.

 

•         Possible Error Messages: Vehicle series input box → Vehicle series is required
