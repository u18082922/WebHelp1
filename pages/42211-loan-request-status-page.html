4.2.21.1 Loan Request Status Page

Description

This page allows the user to be able to view the loan request which were made by the client. The user can then proceed to either accept or reject the loan request.

 

Reference:

Control:

Functionality Description:

1

Loan request link

Click here to view the loan request homepage.

2

Show a number of entries

 

Click here to show only a certain number of entries.

 

3

Search Loan request Input box

Click here to enter the loan request being searched for.

4

Approve button

Click here to approve the loan request

5

Reject button

Click here to reject the loan request.

6

Username Link

Click here to sign out of the website/View profile

7

Return home button

Click here to redirect to the home page of the system
